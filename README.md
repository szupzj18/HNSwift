# HNSwift Client

A hacker news client by Chris.

Fully written in Swift UI.

you can now [install beta test](https://testflight.apple.com/join/6WBYaKWs).

## Feature

- pull the latest top 10 HackerNews posts.
- readmode (backed by safari reader)
